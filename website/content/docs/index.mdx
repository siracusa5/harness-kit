# Harness Kit

Every AI setup starts from scratch. New machine, new project, new teammate — and you're rebuilding configuration you already built somewhere else.

**harness-kit** is a harness-agnostic framework for AI coding tools. Your plugins, skills, MCP servers, hooks, and conventions all live in one portable config. It works with Claude Code today and is designed to travel across tools.

## Quick install

```bash
/plugin marketplace add harnessprotocol/harness-kit
/plugin install research@harness-kit
```
