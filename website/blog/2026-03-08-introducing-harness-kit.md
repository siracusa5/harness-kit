harness-kit v0.1.0 is out. Five plugins for Claude Code, each packaging a workflow you'd otherwise have to set up from scratch every time.

<!-- truncate -->

## What shipped

- **research** — Process any source into a structured knowledge base
- **explain** — Layered code explanations with 6-section structure
- **data-lineage** — Column-level lineage tracing with SVG diagrams
- **orient** — Topic-focused session orientation
- **capture-session** — Session information capture for daily reflection

## Install

```
/plugin marketplace add harnessprotocol/harness-kit
/plugin install research@harness-kit
```

See the [getting started guide](/docs/getting-started/installation) for details.
